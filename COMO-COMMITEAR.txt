# Cómo hacer un commit y subirlo a GitHub
#
# 1) Guarda tus cambios en el editor.
# 2) Abre PowerShell en la carpeta del proyecto y ejecuta:
#
#    git status
#    git add -A
#    git commit -m "Describe aquí el cambio (ej: arreglar login)"
#    git push origin main
#
# Si git status dice "nothing to commit", no hay cambios nuevos que guardar en Git.
# "git add" con una ruta: usa la ruta real del archivo, por ejemplo:
#    git add apps/web/src/App.tsx
